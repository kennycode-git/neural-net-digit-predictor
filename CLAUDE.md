# CLAUDE.md — Digit-Net Visual Lab

Project context for Claude Code sessions.

## Project Overview

Monorepo: handwritten digit classifier with browser inference (ONNX), explainability heatmaps (backend), and a model card page.

Stack: PyTorch (training) → ONNX → onnxruntime-web (Next.js App Router frontend) + FastAPI backend (Captum explainability).

## Directory Map

```
nn_digit_prediction/        ← project root
  ml/                       ← training, evaluation, artifact generation
  backend/                  ← FastAPI explain service
  frontend/                 ← Next.js 14 App Router app
  .github/workflows/        ← CI (ruff + pytest)
```

## Key Constants (must match Python ↔ TypeScript)

- `PREPROCESS_VERSION = "1"`
- `MNIST_MEAN = 0.1307`
- `MNIST_STD = 0.3081`
- Preprocessing: grayscale → bbox crop (+2px pad) → fit into 20×20 (aspect-ratio preserving) → center in 28×28 → /255 → invert → normalize
- **Bug fixed**: original resize was squashing tall narrow "1"s into fat blocks → predicted as "8". Fixed by preserving aspect ratio in both preprocess.py and preprocess.ts.

## DigitCNN Layer Shapes (for NetworkViz)

```
Input           (1,  1, 28, 28)
conv1+relu      (1, 32, 28, 28)
pool1           (1, 32, 14, 14)  ← exported as "conv1_pool"
conv2+relu      (1, 64, 14, 14)
pool2           (1, 64,  7,  7)  ← exported as "conv2_pool"
flatten         (1, 3136)
fc1+relu        (1, 128)         ← exported as "fc1"
fc2/logits      (1, 10)          ← exported as "logits"
```
